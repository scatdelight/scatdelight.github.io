# 문제
[요세푸스 문제 0](https://www.acmicpc.net/problem/11866)

# 풀이
큐를 이용한 구현 문제이다. N-1개의 앞에 있는 수들을 ```pop```하고 그 숫자들을 뒤로 붙이는 식으로 해서, N번째 수들을 result 함수로 넣은 뒤 형식에 맞게 출력하면 되는 문제이다.

# 소스 코드
```c++
#include <iostream>
#include <queue>
#include <vector>

using namespace std;

queue<int> q;
vector<int> result;

void func(int a){
    if(a==1){
        result.push_back(q.front());
        q.pop();
    }
    else{
        int temp = q.front();
        q.pop();
        q.push(temp);
        func(a-1);
    }
}
int main(){
    int N;
    scanf("%d", &N);
    
    
    
    for(int i = 1; i<=N; i++){
        q.push(i);
    }
    
    int M;
    scanf("%d", &M);
    
    while(q.size()>0){
        func(M);
    }
    
    
    
    printf("<");
    for(int i = 0; i<result.size()-1; i++){
        printf("%d, ", result[i]);
    }
    printf("%d>\n", result[result.size()-1]);
}
```
